## Brief #1: Blog ‚Äî Asset Deep Dives ‚≠ê

### Overview

| Field | Detail |
|-------|--------|
| **Pillar** | Asset Deep Dives |
| **Channel** | Blog (avantgarde.finance/insights) |
| **Frequency** | 2x/month |
| **Length** | 800-1200 words |
| **Approver** | CEO |

### Purpose

Provide comprehensive, data-backed risk analysis of individual DeFi assets. Demonstrates the framework's depth and positions Avantgarde as the authority on DeFi asset risk.

### Target Audience

| Audience | What They Want |
|----------|----------------|
| üèõÔ∏è Protocols | Due diligence before treasury integration |
| üè¶ Institutions | Board-ready risk assessment |
| üó≥Ô∏è DAOs | Evidence for governance proposals |

### Content Structure

```
1. HEADER
   - Asset name + ticker
   - Final grade (prominent)
   - One-line verdict

2. EXECUTIVE SUMMARY (100 words)
   - Grade + key finding
   - Primary risk factor
   - Primary strength

3. ASSET OVERVIEW (150 words)
   - What is this asset?
   - How does it work?
   - Key use cases

4. GRADE BREAKDOWN (400-500 words)
   - Score per category (6 categories)
   - Top 2-3 risk factors (with data)
   - Top 2-3 strengths (with data)
   - Any circuit breaker flags

5. DATA SOURCES (100 words)
   - List of on-chain sources
   - API sources used
   - Verification note

6. CONCLUSION (100 words)
   - Final assessment
   - Who this asset is suitable for
   - Key monitoring points

7. CTA
   - Link to dashboard (when public)
   - Link to methodology doc
```

### Tone & Voice

| Do | Don't |
|----|-------|
| Professional, authoritative | Overly academic |
| Data-first, cite sources | Make claims without evidence |
| Accessible explanations | Assume deep DeFi knowledge |
| Balanced (risks AND strengths) | Pure FUD or pure shilling |

**Voice match:** Align with Avantgarde's "curator" positioning ‚Äî institutional rigor, but not cold. Thoughtful expertise.

### Key Messages to Include

- *"The due diligence you'd do ‚Äî if you had 100 hours"*
- Emphasize on-chain verification
- Reference the 6-category methodology
- Note any circuit breaker triggers

### SEO Keywords

`[asset name] risk analysis`, `[asset name] risk score`, `DeFi risk assessment`, `[asset name] grade`, `is [asset name] safe`

### Sample Titles

- "wstETH Risk Analysis: Why It Earns a Grade A"
- "cbBTC Deep Dive: A Comprehensive Risk Assessment"
- "WBTC Under the Microscope: Full Risk Breakdown"

### Assets to Brief First

1. wstETH (Lido) ‚Äî likely A grade, flagship example
2. cbBTC (Coinbase) ‚Äî institutional interest
3. WBTC ‚Äî comparison/contrast potential
4. RLP ‚Äî if differentiated story

### Success Metrics

| Metric | Target |
|--------|--------|
| Time on page | >3 minutes |
| Scroll depth | >70% |
| Shares | 10+ per post |
| Inbound leads | 2-3 per post |
